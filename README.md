# metaverse_frontend

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev:weapp
```

### Compiles and minifies for production
```
yarn build:weapp
```

### Compression and previewing, but the compilation speed will decrease.
```
NODE_ENV=production taro build --type weapp --watch # Mac
```
